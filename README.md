# Interaction patterns emerge from the interplay among abundance, behavior and spatial constraints in mutualistic networks

Mutualistic interactions between species form complex networks that shape evolutionary and ecological processes. Traditionally, these networks have been studied at the species level, with species abundance and traits proposed as key structuring mechanisms. However, such approach often fails to explain commonly observed topological patterns. This limitation may stem from overlooking fine-scale interactions between individuals, the actual network building blocks. By shifting the focus from species to individuals, here we aim to better understand the mechanisms shaping complex mutualistic networks using plant--pollinator assemblages. To do so, we combined individual-based field data and agent-based models (ABMs) within a Bayesian framework. We found that explaining a key network structural pattern, the pollinator community degree distributions, required accounting for species abundances, distinct pollinator behaviors, and spatial plant configurations. Notably, these factors explained a substantial portion of the degree distribution variation even without including specific trait-matching mechanisms. Moreover, our predictive ability varied across different phenological stages, highlighting that distinct mechanisms operate across time. The integration of empirical data with individual-based modeling allowed us to capture details of individual interactions occurring in the real world that revealed how fine-scale processes shape community-wide network structure, offering new insights into the emergence of complex ecological networks.


All data and code necessary to replicate the results of this study are available in this repository. The /Model folder contains all raw interaction data (/Model/Data) and Python code with agent-based models, the /Output_model folder contains the results from the simulation models, the /Analysis_code folder contains the code necessary to replicate all results and figures.
