# Mutualistic networks emerge from the interplay of abundance, behaviour and spatial constraints

**Data and code**:

All data and code necessary to replicate the results of this study are available in this repository. The /Model folder contains all raw plant–pollinator interaction data (/Model/Data) and Python code with agent-based models (a simulation can be generated using the *main_slicesv2.py* file). The /Output_model folder contains the results from the simulation models and any necessary data to perform subsequent analyses. All results and figures can be replicated with the code in the /Analysis_code folder. R code was created with *R version 4.3.1 (2023-06-16)*.
